## üí° The Solution

This project introduces a **temporary, QR-based file sharing system** tailored specifically for print and xerox shops.

Instead of exchanging phone numbers or using chat apps, file sharing becomes **session-based, purpose-driven, and short-lived**.

### How it works

1. The shop generates a **QR code**
2. The customer scans the QR code
3. A temporary sharing session opens instantly
4. The customer shares files, text, or instructions
5. The shop receives files with **unique, organized names**
6. Once the job is completed:
   - the session auto-expires, **or**
   - the shop owner deletes it manually

No follow-ups. No leftover data.
